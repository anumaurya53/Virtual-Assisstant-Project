# Virtual Assistant Project

This project is a virtual assistant that performs tasks such as answering questions, opening applications, and responding to voice commands.

## Features
- Voice command recognition
- Text-to-speech response
- Basic task automation

## Technologies Used
- Python
- SpeechRecognition
- Pyttsx3

## How to Run
1. Install required Python libraries
2. Run the main Python file
3. Give voice or text commands
